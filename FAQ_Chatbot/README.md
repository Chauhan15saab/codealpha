# FAQ Chatbot

## Description

This project is a simple FAQ chatbot developed using Python and Streamlit.

The chatbot uses Natural Language Processing (NLP) techniques to match user questions with stored FAQs using TF-IDF Vectorization and Cosine Similarity.

---

## Features

- User-friendly interface
- NLP-based text preprocessing
- TF-IDF Vectorization
- Cosine Similarity Matching
- Fast and accurate FAQ retrieval

---

## Technologies Used

- Python
- Streamlit
- Pandas
- NLTK
- Scikit-learn

---

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/FAQ-Chatbot.git
```

Go to project folder

```bash
cd FAQ-Chatbot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
